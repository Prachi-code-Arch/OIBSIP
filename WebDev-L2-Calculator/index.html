Calculator

A calculator built with HTML5, CSS3, and vanilla JavaScript — no frameworks, no eval().

Features

  • Digit input, decimal point, and all four basic operators (+, −, ×, ÷)
  • Chained calculations (e.g. 5 + 3 + 2 =) and mid-expression operator switching
  • Clear (C) and Backspace
  • Division-by-zero handled gracefully with an on-screen error message
  • Floating-point rounding correction
